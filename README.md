# Coffee Sales Performance Dashboard | Power BI

## Problem Statement
In retail businesses, understanding sales patterns is essential for improving operations and maximizing revenue. This project develops an interactive Power BI dashboard to analyze coffee shop sales data, focusing on revenue trends, customer behavior, and high-demand time periods.

---

## Objectives
- Examine overall sales performance using transaction-level data  
- Identify time periods with highest customer activity  
- Compare sales behavior between weekdays and weekends  
- Analyze product demand across different coffee categories  

---

## Dataset
- Coffee Shop Sales Dataset  
- Includes fields such as Date, Time, Product, Sales, and Transaction ID  
- Represents daily sales transactions across multiple time intervals  

---

## Tools and Technologies Used
- Power BI  
- Power Query  
- DAX  
- Microsoft Excel  

---

## Approach
- Cleaned and transformed raw data using Power Query  
- Created key performance measures using DAX (Total Revenue, Total Orders, Average Value)  
- Built an interactive dashboard with slicers and filters  
- Designed visuals to highlight trends across time and product categories  

---

## Key Insights
- Total Revenue reached ₹112K  
- Highest sales observed at 10 AM, indicating peak morning demand  
- Weekday sales outperformed weekend sales  
- Latte and Cappuccino emerged as the most popular products  

---

## Dashboard Features
- KPI indicators showing revenue, order count, and average sales  
- Time-based analysis through hourly and daily trends  
- Product-level comparison across different categories  
- Interactive filtering for deeper data exploration  

---

## Business Impact
- Assists in better inventory management based on product demand  
- Helps optimize workforce allocation during peak hours  
- Supports strategic decisions in pricing and promotions  
- Enhances operational efficiency through data-driven insights
