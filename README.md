# Coffee Sales Analytics Dashboard

## Project Overview
This project analyzes coffee shop sales data using Power BI to identify revenue trends, peak sales hours, and customer purchasing behavior.

## Tools Used
Power BI  
Power Query  
DAX  
Excel  

## Key Insights
• Total revenue: ₹112K  
• Peak sales hour: 10 AM  
• Weekdays generated higher sales than weekends  
• Latte and Cappuccino were top-selling products
